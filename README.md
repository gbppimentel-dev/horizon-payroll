# Horizon Payroll Dashboard

A fictional payroll management dashboard created as a movie prop for film production.

This project simulates a corporate payroll system used by the fictional company **Horizon**. It is designed purely for on-screen use in a film and does not connect to any banking systems, payment providers, payroll services, or real employee data.

---

## Disclaimer

This application is a visual simulation only.

- No real payroll processing occurs.
- No money is transferred.
- No banking integrations exist.
- No employee information is stored outside the browser.
- All notifications and payment confirmations are simulated for cinematic purposes.

This project was created solely as a film prop.

---
